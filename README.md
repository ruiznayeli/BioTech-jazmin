# BioTech Dashboard – Agricultura Sustentável

## Descrição do Projeto

O BioTech Dashboard é uma página web desenvolvida com foco na conscientização sobre agricultura sustentável e uso eficiente dos recursos naturais. O projeto apresenta informações relacionadas ao monitoramento da umidade do solo, irrigação inteligente e práticas sustentáveis que podem contribuir para uma produção agrícola mais responsável.

A aplicação foi criada como uma interface visual simples e organizada, permitindo apresentar conceitos importantes da agricultura sustentável por meio de um painel informativo e de fácil navegação.

## Objetivos

* Demonstrar a aplicação de tecnologias web na área da agricultura.
* Promover a conscientização sobre o uso racional da água.
* Apresentar práticas sustentáveis para o manejo do solo.
* Desenvolver uma interface moderna, organizada e acessível.
* Aplicar conhecimentos de desenvolvimento web utilizando HTML e CSS.

## Como o Projeto Foi Desenvolvido

O projeto foi desenvolvido utilizando o Visual Studio Code (VS Code) como ambiente de desenvolvimento.

### HTML5

O HTML foi utilizado para criar toda a estrutura da página, organizando os conteúdos em diferentes seções:

* Cabeçalho com o título e a descrição do sistema.
* Área de configuração contendo campos relacionados à umidade do solo e condições climáticas.
* Painel de status para exibição de informações do sistema.
* Seção com dicas de práticas sustentáveis.
* Rodapé com informações do projeto.

### CSS3

O CSS foi utilizado para definir o design e a aparência visual da aplicação, incluindo:

* Organização dos elementos em cartões (cards).
* Definição de cores relacionadas à sustentabilidade e ao meio ambiente.
* Espaçamentos e alinhamentos para melhorar a legibilidade.
* Estilização de botões, caixas de informação e formulários.
* Layout responsivo para melhor visualização em diferentes dispositivos.

### Ferramentas Utilizadas

* Visual Studio Code (VS Code)
* HTML5
* CSS3
* Navegador Web para testes e visualização

## Funcionalidades

* Interface para visualização de informações sobre irrigação.
* Exibição de dados relacionados à umidade do solo.
* Apresentação de práticas sustentáveis para a agricultura.
* Design simples, intuitivo e organizado.
* Estrutura preparada para futuras implementações e melhorias.

## Conclusão

O BioTech Dashboard foi desenvolvido como um projeto educacional para demonstrar a utilização de HTML e CSS na criação de interfaces voltadas para temas ambientais. O projeto evidencia como a tecnologia pode ser utilizada para divulgar informações importantes sobre sustentabilidade e agricultura, contribuindo para a conscientização e preservação dos recursos naturais.
